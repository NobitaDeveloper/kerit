Frequently Asked Questions

What is KERiT?

KERiT is a powerful Discord bot offering advanced antinuke protection, automod features, and a wide range of interactive tools for server management and community engagement.

How do I invite KERiT to my server?

Visit the homepage and click the "Invite KERiT" button or use the invite link: Invite KERiT.

Is KERiT free to use?

Yes, KERiT offers a comprehensive set of free features, with premium options available for advanced functionalities. Check our support server for details: Join Support Server.

How do I configure antinuke protection?

Use commands like antinuke enable, antinuke config, and specific antinuke settings (e.g., antinuke antiban). Visit the Commands page for a full list.

Can I customize welcome messages?

Yes, use commands like greet message add, greet embed, and greet image add to create custom welcome messages, embeds, and images.

How does the ticket system work?

Set up a ticket panel with ticket panel setup, and users can create tickets via reactions. Manage tickets with commands like ticket close or ticket rename.

What are the automod features?

KERiT's automod includes anti-invite, anti-link, anti-spam, caps lock filtering, and bad word detection. Configure with automod config and related commands.

How can I get help with setup?

Join our support server for real-time assistance from our team and community: Join Support Server.
